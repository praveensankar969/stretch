Privacy — the short version

We don't have any of your data,
because we never collect it.

What Stretch stores

A single JSON file on your computer:

  • Windows: %APPDATA%/Stretch/config.json
  • macOS: ~/Library/Application Support/Stretch/config.json

It contains: your chosen interval, daily goal, quiet-hours, the number of stretches you've completed each day for the last few months, and your current streak.

That's it. It never leaves your device.

What Stretch transmits

  • Update checks — Stretch asks GitHub for the latest release every six hours. GitHub sees your IP address (same as if you visited github.com in a browser). We don't operate any server that sees this.
  • Nothing else. No analytics, no crash reporter, no telemetry, no remote feature flags, no A/B tests, no remote fonts or scripts, no images from a CDN.

The app NEVER

  • Reads your screen, keystrokes, or microphone.
  • Knows which app is in the foreground (it only asks "are you in a full-screen app" so it can stay silent).
  • Phones home with your stretch history or streak.

Removal

Windows: Uninstall Stretch from Windows Settings → Apps. To also erase your settings and history, delete %APPDATA%/Stretch/.

macOS: Drag Stretch from Applications to Trash. To also erase your settings and history, delete ~/Library/Application Support/Stretch/.

Changes

If this ever stops being true we'll say so in the in-app update dialog before shipping. This page is versioned with the source: see PRIVACY.md in the repo for history.
